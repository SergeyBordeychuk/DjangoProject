# Проект 'DjangoProject'
 
## Описание:

Проект DjangoProject - это проект по обучению основам фреймворка django.

## Установка:

1. Клонируйте репозиторий:
```commandline
git clone aaa
```
2. Установите зависимости:
```commandline
pip install -r requirements.txt
```

## Функционал
1. Запуск сервера
```commandline
python manage.py runserver
```
2. Закрыть сервер
